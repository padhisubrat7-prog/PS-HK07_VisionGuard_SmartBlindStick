# PS-HK07_VisionGuard_SmartBlindStick
Smart Blind Stick using ESP32/Nordic MCU with 2 Ultrasonic Sensors, 1 IR Sensor, and GPS Module for real-time obstacle detection (2–3m). Includes buzzer, vibration motor, LED, and SOS push button. Powered by rechargeable battery with charging module. Programmed in Embedded C/Arduino IDE for safe, low-cost mobility assistance.
